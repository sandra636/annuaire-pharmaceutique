# Annuaire Pharmaceutique Régional 💊

Site local présentant les médicaments disponibles dans les pharmacies
dans la ville de douala. Il permet un accès facile aux médicaments et évite les
déplacements inutiles.

---

## 🚀 Installation

1. Cloner le dépôt
```bash
   git clone https://github.com/sandra636/annuaire-pharmaceutique
```
2. Lancer WAMP
3. Copier les fichiers dans `C:/wamp64/www/annuaire-pharmaceutique`
4. Accéder à `http://localhost/annuaire-pharmaceutique`
5. Installer WordPress et activer le thème enfant
